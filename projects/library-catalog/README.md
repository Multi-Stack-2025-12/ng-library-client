# library-catalog

Angular domain library for catalog module use cases, contracts, and adapters used by `ng-library-client`.

## Build

From `app.client/client/ng-library-client`:

```bash
ng build library-catalog
```

Artifacts are generated under `dist/library-catalog`.

## Publish

From `app.client/client/ng-library-client`:

```bash
cd dist/library-catalog
npm publish
```

## Tests

From `app.client/client/ng-library-client`:

```bash
ng test
```

## Notes

- This package is consumed by the app and may depend on local/internal `@eac-arch/*` dependencies.
- For local workspace builds, switch refs using `app.client/client/tools/client-cli.sh refs-project`.
